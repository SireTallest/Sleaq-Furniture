# Sleaq: Crafting Spaces, Creating Memories

## Overview
Sleaq is a modern, responsive web application designed to showcase exquisite furniture craftsmanship. It features an engaging hero section with a dynamic slider, an interactive gallery of furniture designs, and a "Load More" functionality to enhance user experience. Built with a focus on elegant design and smooth interaction, Sleaq brings high-quality furniture visuals to life through a seamless user interface.

## Live Demo
🔗 [https://sleaq-furniture.vercel.app/](#) *still some touches going on...*

## Features
- **Dynamic Hero Slider**: Engages visitors immediately with a captivating carousel of lifestyle imagery.
- **Interactive Gallery**: Showcases a wide range of furniture through a grid layout with lightbox integration for detailed viewing of images and videos.
- **"Load More" Functionality**: Improves performance and user experience by progressively loading gallery items on demand.
- **Responsive Design**: Ensures an optimal viewing experience across various devices, from desktops to mobile phones.
- **Smooth Navigation**: Intuitive navigation bar with anchor links for easy access to different sections of the site.

## Getting Started
To get a local copy of Sleaq up and running, follow these simple steps.

### Installation
1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/SireTallest/sleaq-furniture.git
    ```
2.  **Navigate to the Project Directory**:
    ```bash
    cd Sleaq-furniture
    ```
3.  **Open in Browser**:
    Simply open the `index.html` file in your preferred web browser. Since this is a static frontend project, no server setup is required.
    ```bash
    # Example for Windows
    start index.html

    # Example for macOS
    open index.html
    ```
       Or simply drag `index.html` into any browser.

## Usage
Sleaq offers a straightforward and intuitive user experience for exploring furniture designs.

### Navigating the Site
The navigation bar at the top allows you to quickly jump between sections:
*   **Home**: Returns to the main hero section.
*   **Gallery**: Displays a collection of our crafted spaces.
*   **About Us**: Takes yu to the about us page were you get to know how sleaq came to existence and the purpose they serve.
*   **Testimonials**: Directs you to the reviews/testimonials of previous and current customers who are satisfied with the works of sleaq.
*   **Contact Us**: Directs you to the contact section to get in touch.

### Exploring the Gallery
The **Gallery** section is the heart of Sleaq, showcasing our diverse portfolio.
*   **Image and Video Lightbox**: Click on any image or video thumbnail in the gallery to open an immersive lightbox viewer. This allows you to view high-resolution images and play videos directly within the page.
*   **"Load More" Button**: Initially, only a selection of gallery items is displayed. Click the "Load More" button at the bottom of the gallery to reveal additional furniture pieces and projects, expanding your viewing options.

### Hero Section Slider
The hero section features a dynamic image slider that automatically transitions between different inspiring furniture setups every few seconds, giving a quick overview of Sleaq's aesthetic.
